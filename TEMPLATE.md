# Template: SaaS Starter

Launch your subscription business. Auth, billing, dashboard — everything you need to start charging customers.

## Features to build

- User signup & login
- Stripe subscriptions
- Dashboard with usage stats
- Admin panel
- Welcome emails
- Settings page

## Instructions

This project was created from the "SaaS Starter" template. When the user asks you to build this project, implement all the features listed above. Use Next.js (App Router), TypeScript, Tailwind CSS, and Supabase. The Supabase database, Vercel deployment, and GitHub repo are already set up.
